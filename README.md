# SumsToN

Recursive SumsToN implementation written in Java

## Compile

javac SumsToN.java

### Run

java SumsToN

#### Usage

Enter any positive integer

##### Result

All combinations of positive integers from 1-N which sum to N, will be displayed in non descdening order from left to right.
